PROJECT OVERVIEW
================================================================================

Pi Builder is a beautiful, open-source, MIT-licensed alternative to Auto Maker
that provides unrestricted multi-platform code generation and automation.

Why Build This?
  ‚Üí Auto Maker's license is RESTRICTIVE (custom license blocks monetization)
  ‚Üí We built an OPEN alternative with MIT license (full commercial freedom)
  ‚Üí Better architecture (modular monorepo vs single Electron app)
  ‚Üí Multi-platform from day 1 (web, desktop, mobile, CLI)

AUTO MAKER'S LICENSE PROBLEM
================================================================================

"Monetization" means any activity that generates revenue...including:
  - Reselling or redistributing
  - Including in products/services you sell
  - Hosting as a service
  - Extracting/reselling prompts
  - Creating derivative works for commercial purposes

"Monetization does NOT include: Using for internal organizational use"

‚Üí Translation: You can use it internally, but can't build anything commercial
   on it without getting all 4 Core Contributors to unanimously agree.

SOLUTION: Pi Builder with MIT License
  No restrictions whatsoever.
  Do whatever you want, however you want.
  Commercial freedom from day one.
